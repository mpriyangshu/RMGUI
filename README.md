# RMGUI — Relative Permeability / Fraction Flow (Streamlit)

This repository contains a Streamlit web app that computes relative permeability (krw, kro) and fractional flow (fw) vs water saturation (Sw).  
Adapted from a PyQt GUI version for classroom demonstration.

## Files to add
- `app.py`            — Streamlit app (main)
- `requirements.txt`  — Python dependencies
- (optional) `sample_project.json` — example saved project
